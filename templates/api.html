Passphrasify API

We maintain an API endpoint for generating multiple passphrases. Need enough for all your students?

No problem.

https://passphrase.taggart-tech.com/api/pwlist?n={{ n }}&sep={{ sep }}&digitMin={{ digit_min }}&digitMax={{ digit_max }}

Parameters

n: Number of passphrases (limit 100)

sep: Separator between sections

digitMin: Minimum random digit

digitMax: Maximum random digit

We rate-limit API requests to 1 per minute for load purposes. It's free, after all!
